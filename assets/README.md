# TRNG Assets Folder

This folder contains resources for the TRNG application:

## Contents

- `default_wordlist.txt` - Default word list for passphrase generation (one word per line)
- `icons/` - Application icons (add your .ico files here)

## Customization

### Word List
Replace `wordlist.txt` with your own word list. Format:
- One word per line
- Plain text file (UTF-8)
- Recommended: 1000+ words for good entropy

### Icons
Place application icons in the `icons/` subfolder:
- `app.ico` - Main application icon
- `tray.ico` - System tray icon (if implemented)
